WHAT NOT TO DO

Guardrails that keep the project special

Local-first is a product feature, not a slogan.
Networked models/services stay opt-in, named and removable. A user's meshes, takes and work files are never the price of admission.

Don't become a generic 3D package

A full scene graph, arbitrary nodes, compositing and a new renderer would dilute the edge. Import props into Stage; invest in character performance and portable results.

Don't make WASM a religion

Great for hot loops and battle-tested numerics. It doesn't fix unclear UX, missing project state, or weak cancellation. Benchmark, isolate, always keep a JS fallback.

Don't make AI the only path

Any model output must become editable regular data — selections, landmarks, keys, masks, takes — never an opaque locked result. The deterministic manual tool stays viable.
