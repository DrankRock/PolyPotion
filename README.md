# PolyPotion

## What this is 
 - A suite of tools, made for meshes, to rig them, test them, mocap them, edit them, uv unwrap them etc. All I could think of, I put in.
 - A project, made by Claude Opus 4.8 and Claude Fable 5, using the Design tab. 
 - Something to be shared, not to be stolen.
 - An open project, for anyone to add modules, add characters/animations, fix bugs.
